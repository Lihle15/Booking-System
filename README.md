# EmpowerHER Health Group

EmpowerHER Health Group is a women's healthcare management web application developed using ASP.NET Core MVC. The system is designed to streamline healthcare services by allowing patients to book appointments, manage their medical information, and access essential women's healthcare services through a user-friendly platform.

## Project Overview

This application was developed as part of an Application Development module. It focuses on improving the management of women's healthcare services by providing an efficient and secure digital solution for patients and healthcare professionals.

## Features

- Patient Registration and Login
- Doctor Management
- Appointment Booking
- Appointment Management
- Medical Record Management
- Healthcare Services Information
- Responsive User Interface
- CRUD Operations for all major entities
- Entity Framework Core Database Integration

## Getting Started

### Prerequisites
- Visual Studio 2022
- .NET 8 SDK
- SQL Server
- SQL Server Management Studio (Optional)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Lihle15/Booking-System.git
```
2. Open the solution in Visual Studio.
3. Update the database connection string inside `appsettings.json`
4. Apply Entity Framework migrations
```powershell
Update-Database
```
5. Run the project.
